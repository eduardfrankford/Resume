Development of a Chatbot for Assessing Code Understanding in Artemis

Introduction: This bachelor thesis aims to integrate a chatbot into the Artemis programming platform that simulates tutorial discussions by generating relevant questions based on student submissions to verify individual code understanding.

Methodology:

  • Backend Development: Design services that analyze submissions, generate questions, and evaluate student responses.
  • Frontend Integration: Implement an intuitive Artemis interface enabling interactions between students and teachers.
  • Evaluation: Conduct example exercises and incorporate instructor feedback to assess usability and pedagogical value.
